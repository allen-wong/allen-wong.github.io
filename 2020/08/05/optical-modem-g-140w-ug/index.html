吉比特G-140W-UG优化配置指南

发表于 2020-08-05 更新于 2020-08-13 分类于 光猫配置

实在受不了移动宽带的下载速度。6月底说服老爸把家里宽带换成联通。然而在我妈刷抖音的流量在7月一下子超了14个G之后。。。坑爹的发现三大运营商所有新办套餐没有不限量套餐了！！！这要是换了别家的人估计要被骂成狗。不过，联通给的这猫拿超密实在是太简单了，为联通点赞。

其实就是CUAdmin，账号密码相同。只是界面上的反人类交互我是连蒙带猜试了半天才进去。如果进不去，可以试试用标签上的普通账号登录进去，在用户管理界面右键->检查（这是chrome，其它浏览器自己百度）查看框架源代码，在源代码里查找admin。双引号括起来的含有admin的字符串试一下，各地设备不尽相同。

下面正式开始。

查看超级管理员的账号密码

此步骤如果已经得到超级管理员的账号密码可以省，在网上看北京联通的默认账户名与密码均为CUAdmin。如果密码不对可以再来尝试此步

  1. 使用chrome浏览器进入http://192.168.1.1/
  2. 输入光猫背部的账号与密码登录
  3. 进入管理-用户管理
  4. 在浏览器页面中点击右键，然后点击审查元素
  5. 点击下面的任意元素按ctrl+f(mac下为command+F)搜索login_supcfg
  6. 搜索后你会得到var login_supcfg = { Password:’CUAdmin’, UserName:’CUAdmin’ };这样一个文本，即账号为CUAdmin，密码为CUAdmin
  7. 注销登录

登录超级用户

  1. 使用chrome浏览器进入http://192.168.1.1/
  2. 在登录按钮上(>>符合)右键点击审查元素，然后找到<form id="loginForm" method="post" action="login.cgi">
  3. 双击action="login.cgi"，将action="login.cgi"改为action="cu.html"
  4. 关闭审查元素的小窗口，输入第一步得到的账号密码进行登录,然后成功进入后台

修改至桥接模式

  1. 进入基本配置->上行线路配置，WAN连接列表选中2_INTERNET_B_VID_3961
  2. 将封装类型改为PPPoE，连接模式由路由模式改为桥接模式即可，最后点击保存/启用。之后就需要拔号上网了。
  3. 宽带的账号在办理宽带时打印的给你的业务协议单上可以找到。我这里是区号+8位数字，看起来像座机号，但区号后面是3个0。至于密码，我用联通手厅改宽带密码重试了N次都不行，后来打客服电话让重置密码后，终于可以拨通了，这和网上说的一样。

说一下经验之谈，如果有宽带专线客服，建议打宽带专线，毕竟节省时间也是珍惜生命。联通宽带的默认密码其它地区我不清楚，湖北是123456，建议在一通重置密码操作之前，先试一下。
