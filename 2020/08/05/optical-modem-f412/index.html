中兴F412 V6.0优化配置指南

发表于 2020-08-05 更新于 2020-08-15 分类于 光猫配置

2020年7月初远程帮表哥家看娃的作业打不出来。升级office发现他家带宽只有90M+，逐问之，答曰电信师傅说光猫路由全要换，光猫要价300大洋。于是就有了后来的闲鱼淘来半新F412在家按下文步骤测试好再去他家换猫的经历。一路非常顺利，好文怕丢，转载之，侵删。感谢原作者的分享。原图不喜欢，再加上折腾一次两三个小时，下次有机会再补图吧。

概述

有经验的小伙伴看概述就行了

  1. 拔光纤
  2. http://192.168.1.1/hidden_version_switch.gch，选默认再选湖北（其它的自己选对应位置，如果提示密码错误，则输入telnet拿到的telecomadmin的密码）
  3. 插光纤，注册LOID
    配置丢失用1，2，3，否则直接4，5，注意拔光纤
  4. 拔光纤，复位
  5. telnet 192.168.1.1 用户名root，密码Zte521
  6. sendcmd 1 DB p DevAuthInfo
  7. 观察密码
  8. 得到密码了老鸟们还需要我说么？自己进去改参数
  9. 自己权衡是否需要删除TR069

详细步骤

网上常见的教程是直接telnet进去运行：

观察得到telecomadmin的密码。但实际情况就像这样：

那些教程都是2012，2013年的了。后面的电信都关了telnet功能，你根本连不进去。当然，如果你连接成功，就试一下用户名：root，密码：Zte521，登录后试一下上面的命令，仔细观察结果可得到telecomadmin的密码。如果是和上面图中所示一样连接失败那就继续往下看。

记下光猫的各类参数

  1. 观察光猫背面标签，记往IP、帐号、密码，然后用这个帐号密码登陆进去，可以看到，是6.0版本。
  2. 进网络 -> 远程管理，记住LOID。这个非常重要，一定要记清楚
  3. 进状态 -> 网络侧信息 -> IPV4连接信息，记下VLAN，一般来说就是连接名称后面的数字。例如图中的是INTERNET是1139，TR069的是46，IPTV是3141。这个非常重要，错了就上不了网

光猫复位

现在拔掉光纤，拔掉光纤，拔掉光纤，重要的事说三遍！！！

注意：光纤拔掉后，头子不要乱放，上面如果沾了灰，会影响网速。

找个牙签之类的东西，按住光猫复位键10秒。这时光猫会自动重启，等待光猫重启完成

注意：光猫复位键几年前就已经被电信阉割了，不可能真正复位光猫。这里复位后无需重新注册LOID。直接按下面教程就OK了。

得到超级用记密码

如果你的CMD里没有telnet,请百度，可以通过打开/关闭windows功能解决；

运行CMD

输入后出出来一堆信息，找到红圈的位置，记下用户名密码。这里假设是：telecomadmin、123456

修改路由模式为桥接模式

重新用telecomadmin用户登陆进光猫，进去后，你会发现，比刚才的菜单都变多了，这个时候，光猫的功能才全部都出来了

进网络->宽带设置->internet连接，选择一个你自己的连接，例如这里是那个_1351结尾那个

观察刚才的记录值是否正确，如果你不知道自己拨号信息，可以在这里直接看到（找一个页面密码查看工具看）。有时候电信会**的不告诉你拨号密码。

看到那个允许最大用户数没？ 4个！！！ 删除这个连接。

选择新建wan连接（注意，刚才删除后，这里应该下拉只有2项：新建和TR069那个）

修改以下参数，并保存：

  • 模式：桥模式
  • VLAN：改写(tag)
  • VLANID：1351
  • 802.1p：0

好了，至此，你的光猫已经修改成功了。从现在开始，你必须拨号才能上网！

收尾

插上光纤，插上光纤，插上光纤，设置路由拨号，测试上网及全国版IPTV。

搞错了怎么还原

如果你在配置过程中，把VLAN记错了，或者忘了某一步骤，或者IPTV整到不能用了，怎么还原呢。复位光猫是没用滴。。。。那基本就是个假复位。重新注册也注册不上。恢复方法如下：

  1. 拔掉光纤，拔掉光纤，拔掉光纤
  2. 进这个地址：http://192.168.1.1/hidden_version_switch.gch
  3. 选default version
  4. 点submit(如果提示密码错误，则输入前面拿到的telecomadmin的密码)
  5. 光猫自动重启
  6. 重启完成后，再进：http://192.168.1.1/hidden_version_switch.gch
  7. 选相应的地区，例如撸主选的hubei version
  8. 点submit
  9. 光猫自动重启
  10. 插光纤
  11. 进http://192.168.1.1，点设备注册，输入LOID，等待注册成功就恢复了。

注意，每注册一次LOID，telecomadmin的密码都会变，之前破解的就没用了。

恢复后打开IPTV，直到能进去看电视，打开会才会自动下发配置，不要着急，多等会就能看了。一切就绪后再重新开始上面的步骤！

删除电信远程控制

  1. 用telecomadmin用户登陆光猫
  2. 在刚才建桥接的那个页面选TR069
  3. 点修改，修改VLAN ID，或者直接删除。你们发现修改、删除都不让点，都是灰滴。
  4. chrome、360浏览器极速版或360浏览器极速模式，右键元素审查或检查，移动鼠标，页面上会有一个阴影。当你发现阴影覆盖了这个按纽里，停住！！！把横着的三角一层层，点成朝下滴三角。直到阴影指向你需要的位置，看到修改、删除都在这。
  5. 双击按最后那个disabled并删除掉。现在修改按纽能点了（即使是灰的也能点），事实上点了修改也没用，直接改删除吧
